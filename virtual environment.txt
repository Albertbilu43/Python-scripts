    Open the Command Palette: Press Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (macOS).
    Search for "Python: Create Environment": Type "Python: Create Environment" and select the corresponding command from the dropdown.
    Choose Environment Type: Select Venv for a standard Python virtual environment.
    Select Python Interpreter: Choose the Python interpreter you want to use for this virtual environment. VS Code will then create the environment within your project folder (typically named venv or env).
    VS Code will automatically select: this newly created virtual environment as the interpreter for your workspace.
